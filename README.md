# Task Manager Backend API

This is a Node.js + Express + MongoDB backend project built on mobile using Acode and Termux.

## Features
- Add task
- Update task
- Delete task
- Get tasks

## Tech Stack
- Node.js
- Express
- MongoDB

## About Project
I built this project completely on Android mobile using Acode editor and Termux terminal.
